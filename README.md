# 药时管家

药时管家是一款面向个人健康管理的网页应用，用于集中管理日常用药、药品库存、补药计划、复诊安排和用药记录。应用重点解决慢病患者在复诊开药后容易忘记按时服药、重复购药、漏买药或忘记复诊的问题。

## 功能特点

- **总览仪表盘**：展示今日待办、已完成用药、库存风险和复诊倒计时。
- **今日用药**：按时间展示当日药品、剂量和注意事项，支持服药打卡。
- **药品管理**：维护药品名称、适用疾病、每日用量、库存数量和服药时间。
- **补药计划**：根据库存和每日用量自动计算剩余天数、预计用完日期和补药优先级。
- **复诊管理**：记录下一次复诊时间、医院科室和准备事项。
- **用药记录**：统计近 7 天服药完成率、漏服次数和打卡情况。
- **购药记录**：记录补药数量和购买渠道，并同步更新库存。
- **智能健康摘要**：根据库存、漏服和复诊时间生成简明提醒。
- **新用户指引**：首次打开时介绍核心功能区，后续也可以通过“查看指引”重新查看。
- **精致工作台界面**：统一字体、色彩、阴影和轻量动效，让用药、补药和复诊信息更容易扫读。
- **本地保存**：用户修改的数据会保存到浏览器本地，刷新页面后仍可保留。

## 技术栈

- React
- Vite
- Vitest
- Browser LocalStorage
- Phosphor Icons

## 本地运行

```bash
npm install
npm run dev
```

启动后打开终端显示的本地地址即可访问。

## 构建部署

```bash
npm run build
```

构建产物会生成在 `dist` 目录，可部署到 Vercel、Netlify 或其他静态网页托管平台。

## 测试

```bash
npm test
```

测试覆盖药品库存计算、补药计划、复诊状态、用药记录统计和页面状态更新规则。

## 使用说明

1. 首次打开应用时，可以按引导了解总览、用药、补药、复诊和记录功能区。
2. 在“今日用药”中点击“标记服用”，系统会同步更新完成数和药品库存。
3. 在“药品管理”中新增或编辑药品后，补药计划会自动重新计算。
4. 在“购药记录”中添加购买数量后，对应药品库存会自动增加。
5. 在“复诊管理”中添加新的复诊安排后，首页会显示最新倒计时。
6. 需要再次查看页面说明时，可以点击顶部的“查看指引”。
